无需魔法，国内部署可直接访问的chatGPT

需要：
 Github    vercel   cloudflare    chatGPT账号（可以网上找共享的或者自己注册）   域名（子域名）

我部署的镜像网站(看看效果)

开始（如果你有chatGPT账号）

pandora-cloud-serverless 是一个简单的仓库，用于Serverless部署Pandora-Cloud。

点击展开Github主页


一键部署

推荐的方法：

Vercel

Railway

Gitpod

部署

选择你希望使用的部署方法
这里使用Vercel举例
点击上面一键部署的链接
点击展开图片

在一键部署打开的链接中
选择Github登录方便创建 Git 存储库
授权登录到github
点击展开图片

为你的储存库取一个名字
继续

只需等待一会儿即可完成部署
点击张开图片



点击 继续到仪表盘


点击Domains转到域名设置
输入域名后点击add （图1）
选择直接添加  **** （图2）



然后会显示如下


记录下内容
等下进入cloudflare将域名解析至此

子域名

 简单的很
 比如我有一个域名polar-bear.eu.org
 我已经绑定了我的博客
 选择我不想绑定其他域名
 而想加入chatGPT功能到这个域名下
 只需再设置一个子域名chat.polar-bear.eu.org即可
 所以只需要将上面add的域名换成chat.polar-bear.eu.org
 再在cloudflare中的polar-bear.eu.org设置中
 添加一个DNS
 将chat.polar-bear.eu.org定向到cname.vercel-dns.com（vercel上部署的chatGPT网页）
 图：

绑定域名

因为Vercel污染严重
所以需要自己绑定域名用于访问

上次已经说过如何免费注册一个二级域名
点这里跳转

假设现在已经申请好了域名

打开cloudflare

cloudflare

注册或登录后
在主界面添加域名
输入你刚刚注册的域名
选择免费计划




DNS设置可以先跳过等下再设置



更改您的名称服务器


记录下两个名称服务器
然后
进入https://nic.eu.org/
点击张开图片


点击域名进入管理
点击张开图片

点击名称服务器
点击张开图片

将cloudflare刚刚给出的名称服务器输入
保存确定即可
没有错误就成功了


然后回到cloudflare
点击完成检查服务器


快速设置save就行

等到cloudflare成功解析服务器
