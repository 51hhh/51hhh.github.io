创建多个ssh key

在创建ssh key时自行输入路径和文件名称，而非使用默认路径和文件名即可。

或者使用 -f来指定文件名

ssh-keygen -t rsa -C “youremail@email.com“ （-f ~/.ssh/second-rsa）
